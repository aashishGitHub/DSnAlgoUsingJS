## Pattern: Binary Search

**When to use:**
- Sorted arrays (or search space that can be halved)
- Need O(log n) search
- Finding boundaries / position / "can we achieve X?"
- Square root, nth root, median of two sorted arrays

**Key indicators:**
- "Sorted array"
- "O(log n) time"
- "Search" / "Find position"

**Complexity:** Time O(log n) | Space O(1)
