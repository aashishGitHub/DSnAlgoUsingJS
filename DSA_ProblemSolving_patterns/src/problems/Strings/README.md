# Strings Pattern

**When to Use**: String manipulation, palindromes, substring search, string operations  
**Time Complexity**: O(n) to O(nÂ²) | **Space Complexity**: O(1) to O(n)

## Problems in this folder:

1. **reverseString.js** - Reverse a string
2. **findSubstring.js** - Find index of substring
3. **longestPalindrome.js** - Find longest palindrome from array of strings
4. **binaryAdd.js** - Add binary strings

## Common Techniques:

- **Two Pointers**: For palindromes, reversing
- **Sliding Window**: For substring problems
- **Hash Map**: For character frequency
- **String Manipulation**: For basic operations

## Related Patterns:

- Two Pointers (for palindromes)
- Sliding Window (for substring problems)
- Hash Map (for anagrams, frequency)
