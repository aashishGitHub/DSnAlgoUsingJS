# Graph Pattern

**When to Use**: Path traversal, DFS/BFS, graph algorithms  
**Time Complexity**: O(V + E) | **Space Complexity**: O(V)

## Problems in this folder:

1. **traversePath.js** - Path traversal problems
2. **findPath.js** - Find path from source to destination

## Common Techniques:

- **DFS (Depth-First Search)**: Recursive traversal
- **BFS (Breadth-First Search)**: Level-order traversal
- **Backtracking**: For path finding
- **Memoization**: For optimization

## Related Patterns:

- DFS/BFS (Tree Traversal)
- Backtracking
- Dynamic Programming (for path optimization)
